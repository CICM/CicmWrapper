Detailed Description

The ebox part.

This part refers to the methods and structures that can be used by all the t_ebox structures.

Data Structure Documentation

struct t_ebox

The CICM GUI object.

It contains the CICM object with all the members for graphical behavior. This should be used for graphical object that don't have signal processing methods.
