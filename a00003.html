Detailed Description

The edspobj part.

This part refers to the methods and structures that can be used by all the t_edspobj structures.

Data Structure Documentation

struct t_edspobj

The CICM DSP object.

It contains the CICM object with all the members for signal processing. This should be used for no graphical object that have signal processing methods.
